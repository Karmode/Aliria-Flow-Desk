# Aliria Desk 🖥️

El centro de mando digital para la administración de clientes, cotizaciones y cuentas de cobro, diseñado para pequeños negocios de servicios.

## 📝 Descripción

Este proyecto fue creado para mi mamá, Aliria, con el fin de ofrecerle un "escritorio digital" donde pueda organizar toda la gestión de su negocio de mantenimiento. La meta de **Aliria Desk** es centralizar la información que antes estaba en papel o en distintos archivos, proporcionando orden, claridad y control.

El sistema está diseñado para ser la única herramienta que necesite para administrar sus proyectos desde el contacto inicial con un cliente hasta que el pago ha sido recibido.

## ✨ Características Principales

* **Gestión de Clientes:** Base de datos centralizada con la información de los clientes.
* **Generador de Documentos:** Crea cotizaciones y cuentas de cobro profesionales en PDF.
* **Trazabilidad Completa:** Convierte cotizaciones en cuentas de cobro para seguir el ciclo de vida de cada trabajo.
* **Control de Estado de Pagos:** Mantén un registro claro de las cuentas pendientes, pagadas y vencidas.
* **Dashboard de Control:** Panel principal con un resumen visual del estado del negocio.
* **Servicios Predefinidos:** Carga y reutiliza ítems de servicios comunes para no tener que escribirlos cada vez.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** Python Streamlit
* **Backend:** Python
* **Base de Datos:** MongoDB
